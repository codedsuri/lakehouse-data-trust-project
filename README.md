# lakehouse-data-trust-project
A real-world simulation project designed to demonstrate how modern organizations build trusted data products using strong data governance foundations. This project combines Data Quality, Metadata Management, Data Lineage, Business Glossary, Collibra implementation, SQL validations, and BI consumption into one end-to-end ecosystem.

This project draws from my real experience across EY’s ML forecasting ecosystem, Seven West Media’s BI/data quality governance work, and structured Collibra governance implementation to showcase how data becomes a reliable product for business users.
